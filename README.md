# Amazon Clone Project

This is a replica of the Amazon website built using **HTML** and **CSS**. The project focuses on recreating the **layout, header, hero section, and product display sections** of Amazon’s homepage.  
It’s designed for practice and learning purposes to understand web page structuring and styling.

## Features

- Fully responsive **header** with navigation links
- **Hero section** with banners and promotional images
- Product display sections styled similarly to Amazon
- Clean and modern **UI design** using HTML and CSS
- Easy to customize and extend

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, and basic animations)
